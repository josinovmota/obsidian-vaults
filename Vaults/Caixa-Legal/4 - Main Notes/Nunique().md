# Nunique()

A função **nunique()** do pandas tem como objetivo retornar uma **Serie** que conta quantos valores distintos tem dentro do input 

# Example

```python
df = pd.DataFrame({A: [3, 2, 1], B: [3, 2, 2]})

df.nunique()

A   3
B   2
```
