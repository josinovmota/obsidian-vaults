# Spaces around assignments

No python, quando fazer um **assignment** dentro de uma função, não devemos utilizar espaços. Por exemplo:

```python
print(nome="José", idade=30)
```

Mas que ao criar váriáveis podemos sim utilizas os espaçamentos. Por exemplo:

```python
nome = "José"
idade = 30
```
