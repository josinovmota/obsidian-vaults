# H2 DB

H2 DataBase é um banco de dados relacional escrito em java, amplamente utilizado por sua leveza e praticidade

# Dependency do H2 DB no Maven
```xml
    <dependencies>
        <!-- H2 Database -->
        <dependency>
            <groupId>com.h2database</groupId>
            <artifactId>h2</artifactId>
            <version>2.2.224</version>
        </dependency>
    </dependencies>
```
