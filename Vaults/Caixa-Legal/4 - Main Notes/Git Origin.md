# Git Origin

A **Origin** no git funciona como um apelido para o repositório daquele ambiente. Por exemplo, imagine que você está usando o **Whatsapp** e nele você tem que adicionar o seu chefe. No lugar de utilizar apenas o número dele(que é difícil de lembrar) `+55 11 4002-8922` você usa um apelido como `Chefe Trabalho`. O **Origin** funciona da mesma forma. Ele é um apelido para o link do seu repositório. Então no lugar de escrever:

```shell
git push https://github.com/chetachiezikeuzor/cMenu-Plugin main
```

Podemos escrever:

```shell
git push origin main
```
