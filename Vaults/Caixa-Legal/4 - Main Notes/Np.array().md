# Np.array()

O comando np.array() serve para criar uma **Series** ou **DataFrame** baseado em listas.

# Example

```python
df = np.array([[1,2,3], [4,5,6]])

# Esse comando serve pra criar um DataFrame com 2 linhas e 3 colunas

A 1, 2, 3
B 4, 5, 6

```
